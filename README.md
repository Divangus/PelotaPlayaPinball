# PelotaPlayaPinball

## Description

This game is a nice pinball.

## Key Features

 - 1 player
 - Control with the keyboard
 
## Controls

Keyboard
- Left and Right  - Movement
- Down - Start game

### Debug functionality

Keyboard
- F1 - View collisions
- F3 - Instant dead
- C - Add 100 points

### Combos
- After hitting the bonus player gain extra points (Three red rectangles)
